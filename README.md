# Tanzila — Personal Portfolio Website

A clean, responsive portfolio website built with HTML, CSS, and JavaScript.

---

## How to Edit Content

All content is in `index.html`. Each section is clearly commented.

- **Name / Bio** — find the `hero` section, edit the `hero-name` and `hero-bio` tags
- **Education** — find `id="education"`, edit the `card-title`, `card-sub`, `card-desc` inside each `info-card`
- **Experience** — find `id="experience"`, same card structure
- **Skills** — find `id="skills"`, add or remove `skill-tag` spans
- **Projects** — find `id="projects"`, same card structure
- **Contacts** — find `id="contacts"`, update email, phone, and social links

---

## How to Update Images

1. Replace `assets/profile.png` with your new photo
2. Keep the filename exactly `profile.png`
3. Recommended size: 180px × 220px

To update the resume:
1. Replace `assets/resume.pdf` with your new PDF
2. Keep the filename exactly `resume.pdf`

---

## How to Deploy (GitHub Pages)

1. Push the project to a GitHub repository
2. Go to the repo → **Settings** → **Pages**
3. Under Branch, select `main` → click **Save**
4. Your live site will be at: `https://your-username.github.io/repo-name`
